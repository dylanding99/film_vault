# FilmVault 开发路线图

## 当前版本：v0.6.0

### v0.6.0 已完成功能

**设计系统（Design System）**：
- 电影级编辑暗色风格设计语言
- 完整的设计令牌系统（colors, spacing, typography, border, shadow, transition）
- 品牌色系统、背景层级、文本层级
- 胶片框效果、胶片颗粒效果、发光效果
- 按钮系统、卡片系统、筛选器系统、搜索栏系统
- 丰富的动画系统（fadeInUp、slideIn、scaleIn、float 等）

**新增组件**：
- Header 组件 - 统一的页面头部，支持滚动效果
- EmptyState 组件 - 优雅的空状态展示
- Toast 通知系统 - 基于 Sonner 的提示反馈

**工具函数**：
- 相机常量库（130+ 型号，支持按品牌/系列分组）
- 图片加载 Hook（useImageAsset）
- 错误处理 Hook（useErrorHandler）

**UI 一致性提升**：
- 所有组件使用设计令牌
- 统一的对话框尺寸变体
- 统一的图标尺寸
- 自定义滚动条样式

---

## 上一版本：v0.5.0

### 已完成功能

**设计系统基础**：
- 设计令牌系统创建
- Dialog 组件尺寸变体支持
- 图标尺寸统一

**EXIF 模块**：
- ExifInfoPanel 组件 - 显示照片 EXIF 信息（ISO、光圈、快门、焦距、GPS）
- PhotoMetadataForm 组件 - 编辑单张照片的拍摄参数
- 从文件读取 EXIF 数据功能
- 保存 EXIF 到文件功能
- 混合架构：胶卷元数据保留在数据库，照片 EXIF 从文件读取

**筛选与搜索功能**：
- RollFilters 组件 - 按胶卷型号、相机筛选
- 搜索框 - 按胶卷名称、备注搜索
- 收藏筛选（只看有收藏的胶卷）
- URL 参数同步 - 支持分享筛选链接

**设置页面增强**：
- EXIF 设置面板 - ExifTool 状态显示
- 自动写入 EXIF 开关
- 并发写入数设置（1-8）

---

## 上一版本：v0.4.0

### 已完成功能

**位置功能增强**：
- LocationSearchInput 组件 - 城市搜索和位置选择
- 地理编码集成 - 使用 OpenStreetMap Nominatim API
- 位置信息存储 - Roll 表和 Photo 表添加地理位置字段

**筛选与搜索功能**：
- RollFilters 组件
- URL 参数同步
- 收藏筛选

**其他功能**：
- 胶卷导入功能（支持 JPG、PNG、WebP 等格式）
- 自动缩略图生成（300px WebP）
- 自动预览图生成（1920px WebP）
- 胶卷列表展示（卡片式布局）
- 元数据编辑（名称、胶卷型号、相机、镜头、日期、备注）
- 数据持久化（SQLite 数据库）
- 自定义存储位置
- 胶卷详情页面
- 照片收藏功能
- 批量删除功能
- 完整中文化界面
