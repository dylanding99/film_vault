# FilmVault 开发路线图

## 当前版本：v0.3.0

### v0.3.0 已完成功能

**EXIF 模块**：
- ExifInfoPanel 组件 - 显示照片 EXIF 信息（ISO、光圈、快门、焦距、GPS）
- PhotoMetadataForm 组件 - 编辑单张照片的拍摄参数
- 从文件读取 EXIF 数据功能
- 保存 EXIF 到文件功能
- 混合架构：胶卷元数据保留在数据库，照片 EXIF 从文件读取

**筛选与搜索功能**：
- RollFilters 组件 - 按胶卷型号、相机筛选
- 搜索框 - 按胶卷名称、备注搜索
- 收藏筛选（只看有收藏的胶卷）
- URL 参数同步 - 支持分享筛选链接

**设置页面增强**：
- EXIF 设置面板 - ExifTool 状态显示
- 自动写入 EXIF 开关
- 并发写入数设置（1-8）

---

## 上一版本：v0.2.x

### 已完成功能

**核心功能**：
- 胶卷导入功能（支持 JPG、PNG、WebP 等格式）
- 自动缩略图生成（300px WebP）
- 自动预览图生成（1920px WebP）
- 胶卷列表展示（卡片式布局）
- 多胶卷管理（自动路径冲突处理）

**编辑功能**：
- 元数据编辑（名称、胶卷型号、相机、镜头、日期、备注）
- 数据持久化（SQLite 数据库）

**其他功能**：
- 自定义存储位置
- 胶卷详情页面
- 照片收藏功能
- 批量删除功能
- 完整中文化界面

---

## 未来版本规划

### v0.4.0 - 轻量级地理功能增强

**位置编辑优化**：
- 优化 LocationSearchInput 组件的搜索体验
- 添加最近使用位置列表
- 改进错误处理和加载状态
- 支持清除位置快捷操作

**位置显示增强**：
- 优化 ExifInfoPanel 中位置信息的显示
- 添加位置字段为可点击的链接（链接到系统地图应用）
- 在照片详情页显示位置信息（城市、国家、坐标）

**批量操作优化**：
- 改进"应用到所有照片"功能的用户体验
- 添加批量清除位置功能
- 优化位置应用的性能

**数据层优化**：
- 完善地理编码缓存机制
- 添加批量位置更新的事务处理
- 优化位置相关的数据库查询

**EXIF 模块完善**：
- UserComment 格式优化

- 添加 UserComment 格式验证

**EXIF 写入优化**：
- 添加 EXIF 写入进度显示
- 优化并发写入的配置
- 改进 EXIF 写入失败的重试机制

**读取功能优化**：
- 优化 EXIF 读取的性能
- 添加 EXIF 数据缓存
- 改进读取失败的处理逻辑

### v0.5.0 - 高级功能

- 多条件搜索（日期、胶卷型号、相机、评分等）
