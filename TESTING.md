## 测试用例

### 模块 1: 基础设置

| 测试项 | 测试步骤 | 预期结果 | 状态 |
|--------|---------|---------|------|
| 启动应用 | 运行 `npm run tauri:dev` | 应用正常启动，界面显示正常 | |
| 界面显示 | 检查首页 UI | 显示 FilmVault 标题、导入按钮 | |
| 设置图库路径 | 点击设置图标 → 选择路径 → 保存 | 路径已保存，下次启动保持 | |
| ExifTool 状态检查 | 打开设置对话框 | 显示 ExifTool 可用/不可用状态 | |

### 模块 2: 导入功能

| 测试项 | 测试步骤 | 预期结果 | 状态 |
|--------|---------|---------|------|
| 导入第一个胶卷 | 选择文件夹 → 填写信息 → 导入 | 导入成功，新胶卷显示 | |
| 导入进度显示 | 观察导入过程 | 进度条、计数、文件名实时更新 | |
| 导入后立即显示 | 导入完成后观察首页 | 新胶卷立即显示 | |
| 导入空文件夹 | 选择不含图片的文件夹 | 前端提示"未找到照片" | |
