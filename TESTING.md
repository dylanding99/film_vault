## 测试用例

### 模块 1：基础设置

**测试目标**: 验证设置对话框可以正确选择存储位置

**测试步骤**:
1. 启动应用：`npm run tauri:dev`
2. 打开设置对话框
3. 点击"自定义存储位置"
4. 在文件选择对话框中，选择一个存在的目录
5. 确认选择
6. 验证：设置面板显示的存储路径已更新为新选择的路径

**预期结果**:
- 设置面板正确显示新选择的存储路径
- 数据库 settings 表中的 `library_root` 字段被更新
- 重启应用后，存储路径保持不变

---

### 模块 2：胶卷管理

**测试目标**: 验证胶卷导入功能

**测试步骤**:
1. 启动应用：`npm run tauri:dev`
2. 点击"导入胶卷"按钮
3. 在文件选择对话框中，选择一个包含照片的文件夹
4. 确认选择

**预期结果**:
- 文件对话框正常打开
- 图片处理正常进行（生成缩略图和预览图）
- 胶卷成功添加到数据库
- 胶卷列表页面正常显示新导入的胶卷
- 缩略图和预览图文件正确生成到 `[年份]/[日期]_[胶卷型号]_[相机]/thumbnails/` 和 `previews/` 目录

**注意事项**:
- 测试前请确保有可用的照片文件夹
- 导入的文件夹应包含 JPG、PNG 或 WebP 格式的照片
- 导入完成后会自动重命名和整理文件到统一的目录结构

---

### 模块 3：筛选与搜索

**测试目标**: 验证搜索功能正常工作

**测试步骤**:
1. 在胶卷列表页面，点击筛选下拉菜单
2. 按胶卷型号筛选
3. 验证：列表只显示匹配的胶卷
4. 清除筛选条件
5. 按相机型号筛选

**预期结果**:
- 每个筛选条件都能正常工作
- 筛选后立即生效
- 多个筛选条件可以同时生效（如：胶卷型号 + 相机）
- 清除筛选功能正常工作

---

### 模块 4：删除功能

**测试目标**: 验证删除操作的安全性

**测试步骤**:
1. 导入测试胶卷
2. 点击"删除"按钮（选择模式）
3. 选中多个胶卷
4. 确认删除

**预期结果**:
- 删除确认对话框正常显示
- 删除后从数据库移除相关数据
- 缩略图和预览图文件被删除
- 照片数据保留

---

### 模块 5：UI 交互

**测试目标**: 验证按钮响应和交互状态

**测试步骤**:
1. 测试 Hover 状态
2. 测试按钮点击事件
3. 测试选择模式
4. 测试 Toast 通知

**预期结果**:
- 所有按钮 Hover 状态正常
- 点击事件正常触发
- Toast 通知正常显示
- 选择模式切换正常
